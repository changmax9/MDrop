# MDrop 0.2.0 Release Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship MDrop 0.2.0 with the reference Liquid Glass mode-picker animation, polished settings, eight immediately switchable languages, Folded M branding, Sparkle updates, and a verified GitHub DMG release.

**Architecture:** Keep SwiftUI and AppKit responsibilities explicit: SwiftUI owns Settings and Shelf visuals, `AppLanguageController` owns locale state and explicit lookups, AppKit rebuilds native menus when locale changes, and `UpdateService` wraps Sparkle. Brand assets come from one deterministic vector master; packaging stages resources and Sparkle.framework into one ad-hoc signed app bundle.

**Tech Stack:** Swift 6, SwiftUI, AppKit, Observation, String Catalogs, Sparkle 2.9.2, Swift Testing, SwiftPM, Xcode project builds, `iconutil`, `codesign`, `hdiutil`, GitHub CLI.

---
